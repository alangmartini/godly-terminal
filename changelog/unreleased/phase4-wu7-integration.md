### Added
- **Split pane integration** — Ctrl+\ splits horizontally, Ctrl+Alt+\ splits vertically, Ctrl+Shift+\ unsplits, Alt+\ cycles focus between panes
- **Clipboard copy/paste** — Ctrl+Shift+C copies selected text, Ctrl+Shift+V pastes from clipboard into focused terminal
- **Mouse text selection** — click and drag to select text with blue highlight overlay, Ctrl+Shift+A selects all
- **Layout tree rendering** — multi-pane view using `view_layout()` replaces single-pane view
- **Focus routing** — keyboard input routes to focused pane, independent of active tab
